# BoxOffice
> 주간/주말 박스오피스 순위 앱

<br>

## 1. 프로젝트 소개

개발환경 : Xcode 15.4, iOS 17<br>
사용 API : 영화진응위원회(Kobis), The Open Movie Database(OBDb)<br>
채택한 아키텍쳐: MVVM Architecture<br>
기술 스택 : Swift, SwiftUI<br>
주요 기능 : 주간/주말 박스오피스 순위 조회, 영화 상세 정보 확인<br>
